# 🛋️ كتالوج الأثاث المنزلي - Brshour Furniture

هذا المشروع عبارة عن صفحة ويب منظمة وسريعة لعرض منتجات الأثاث (أكثر من 180 موديل) مقسمة إلى أقسام سهلة التصفح. يتميز التصميم بالفخامة (نظام الألوان الداكن مع الذهبي) ويدعم العرض على الموبايل والكمبيوتر بشكل ممتاز.

## ✨ المميزات:
- **تنظيم تلقائي**: يتم سحب البيانات من ملف `data.js` لسهولة التعديل.
- **تصفح ذكي**: أقسام لعرض (وحدات التلفزيون، الجزامات، الأنتريهات، المطابخ، السفرة، غرف النوم، الدواليب).
- **عرض الصور**: خاصية الـ Lightbox لتكبير الصور والتنقل بينها.
- **تواصل واتساب**: زر عائم للتواصل المباشر مع العملاء.
- **سريع جداً**: تم تحسين الصور والأكواد للتحميل السريع.

## 🚀 كيفية الرفع على GitHub Pages:
1. قم بإنشاء مستودع (Repository) جديد على حسابك في GitHub.
2. ارفع جميع الملفات الموجودة في هذا المجلد (`index.html`, `data.js`, وجميع ملفات الصور `.jpg`).
3. انتقل إلى إعدادات المستودع (**Settings**).
4. من القائمة الجانبية اختر **Pages**.
5. تأكد من أن المصدر (Source) هو **Deploy from a branch** واختر فرع `main` (أو `master`) والمجلد `/ (root)`.
6. اضغط **Save**. بعد دقائق قليلة، سيعطيك GitHub رابط موقعك المباشر.

## ✏️ تعديل البيانات (الأسعار والأسماء):
لتعديل اسم منتج، وصفه، أو سعره:
1. افتح ملف `data.js`.
2. ابحث عن كود المنتج أو رقم الصورة.
3. قم بتعديل القيم التالية:
   - `name`: اسم المنتج.
   - `desc`: تفاصيل إضافية.
   - `price`: السعر بالجنيه المصري.
